Popditt is a responsive Reddit clone built with TypeScript, React, and Redux, using Vite as the build tool. The project aims to familiarize the developer with Vite and improve skills in TypeScript, React, Redux, and unit testing. Key features include top Reddit categories, category-specific posts, and individual post pages. The technology stack includes Vite, TypeScript, React, Redux, CSS, React Testing Library, Jest, and the Reddit API. The project is actively maintained, with room for improvement acknowledged. Check out the live demo for a hands-on experience. For more details, refer to the project documentation.

Jamming is a React.js application created by Filip Popendyk, designed for effortless playlist creation with favorite songs using the Spotify API. This project showcases proficiency in React.js, specifically in displaying data from the Spotify API and working with class components. The technology stack includes React.js for building the frontend, HTML for markup, CSS for styling, and integration with the Spotify API for dynamic content. The live version of the app is hosted on Netlify. This project serves as a showcase for React.js skills, API integration, and front-end development using HTML and CSS.

Three Star Media is a website developed to showcase proficiency in HTML and CSS, specifically focusing on Flexbox and Grid layout techniques.
